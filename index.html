Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

  • L = Luas
  • a = Panjang Alas

Rumus Keliling Segitiga yaitu :

K = Sisi A + Sisi B + Sisi C

    Dimana :
  • K = Keliling
  • Sisi A = Panjang Sisi A
  • Sisi B = Panjang Sisi B
  • Sisi C = Panjang Sisi C
